# Plan: vietnam-elections-wikidata

A new repo under `bamboo-filing-cabinet/` to reconcile and create Wikidata items for Vietnamese National Assembly delegates, starting with the 15th NA (2021).

## Goal

Create Wikidata items for ~500 elected NA15 delegates using the structured data already in `vietnam-elections`, then backfill the resulting QIDs into the source dataset.

## Data access

- **Primary:** sibling path `../vietnam-elections/public/data/elections/na15-2021/` when both repos are in the monorepo
- **Fallback:** fetch from `https://bamboo-filing-cabinet.github.io/vietnam-elections/data/elections/na15-2021/`
- A small loader module checks for the sibling path first, falls back to HTTP

## Phases

### Phase 1: Reconciliation

Before creating anything, figure out who already has a Wikidata item.

- Load persons from `results.json` (filter to winners — those with `"won"` in statuses)
- Match against Wikidata by name + DOB using the Wikidata API or SPARQL
  - Vietnamese names need careful handling — diacritics, folded forms, name order
- Produce a reconciliation report: matched (with QID), unmatched (candidates for creation), ambiguous (needs manual review)
- Output: `reconciliation/na15-2021.csv` with columns: `person_id`, `name`, `dob`, `wikidata_qid`, `match_status`, `match_notes`

### Phase 2: Prepare statements

For unmatched persons, prepare Wikidata statements from our data:

- **Labels:** Vietnamese name (vi), folded ASCII name (en fallback)
- **Statements:**
  - P31 (instance of) → Q5 (human)
  - P569 (date of birth) → from `dob`
  - P21 (sex or gender) → from `gender`
  - P27 (country of citizenship) → Q881 (Vietnam)
  - P172 (ethnic group) → from `ethnicity` (needs mapping to Wikidata items)
  - P39 (position held) → Q10841192 (member of the National Assembly of Vietnam), qualified with:
    - P2937 (legislative term) → Q107979136 (15th National Assembly)
    - P768 (electoral district) → the province/city item (existing convention; see verified-identifiers note)
    - P580/P582 (start/end time) → 2021-07-20 / 2026-05-22
  - P1343 (described by source) → link to CEMA bulletin
- Output: `statements/na15-2021-create.qs` (QuickStatements v2 format) or JSON for the Wikidata API

### Phase 3: Upload

- Review prepared statements manually (or sample-check)
- Upload via QuickStatements batch or Wikidata API with bot flag
- Log results: `uploads/na15-2021-log.csv` with `person_id`, `wikidata_qid`, `upload_status`, `timestamp`

### Phase 4: Backfill QIDs

- Merge the reconciliation + upload results into a single mapping: `person_id` → `wikidata_qid`
- Contribute this mapping back to `vietnam-elections` (either as a data file or a PR adding QIDs to the person records)
- Also add ISO 3166-2:VN codes on localities (straightforward — 63 provinces, all have codes, pin to 2021 boundaries)

## Open questions

- **Bot approval:** Partially resolved (2026-07-04). QuickStatements needs an *autoconfirmed* account (≥50 edits, first edit ≥4 days old), not a bot flag. There is no numeric threshold forcing bot approval — only "very large or potentially-controversial runs" go through Wikidata:Bots. Still a judgment call for ~500 creations. See `docs/journals/2026-07-04.02.quickstatements-access-requirements.md`.
- **Constituency items:** ~~Do NA15 constituencies have Wikidata items?~~ Resolved (2026-07-04): no dedicated constituency items are in use; the existing convention points P768 at the province/city item. See verified-identifiers note.
- **Ethnic group mapping:** Our `ethnicity` field has Vietnamese-language values (Kinh, Tày, etc.) that need mapping to Wikidata QIDs. Confirmed the items exist (2026-07-04); a lookup table is feasible, but some values are ambiguous (e.g. "người Thái" has several candidate items) and need manual disambiguation. See verified-identifiers note.
- **Losing candidates:** Defer for now. Notability is unclear, and 500 winners is already a significant contribution.
- **Ongoing maintenance:** When NA16-2026 data is added to `vietnam-elections`, this repo should be reusable for the next cycle with minimal changes.
